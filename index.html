service.ipcdatastore documentation¶

Introduction:¶

service.ipcdatastore is an addon for Kodi for sharing python objects among python addons. It requires that script.module.ipc is also installed.

These addons currently have been tested and are working on:

  1. Windows (Helix B2 and Gotham 13.2)
  2. OSX (Helix B2 and Gotham 13.2)
  3. Android (Helix B2 and Gotham 13.2)
  4. XBMCbuntu (Gotham 13.2)

They may be downloaded and installed from:

  • https://github.com/KenV99/script.module.ipc
  • https://github.com/KenV99/service.ipcdatastore
